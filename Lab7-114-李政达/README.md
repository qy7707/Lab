# Lab7 - 114 李政达

说明：本项目为 Lab7 作业样例，页面文件为 `Lab7-114-李政达.html`，样式文件为 `styles.css`。

作业要求实现的要点：
- 页面包含3个主题区块，每个为3x3九宫格
- 第二主题使用父相子绝（父元素 position:relative；子元素 position:absolute）实现布局
- 顶部主题顶部行图片悬浮放大 1.5 倍；该主题标题 hover 顺时针旋转 45°
- 第二主题图片悬浮倾斜 30°
- 新增固定定位页脚

使用方法：在浏览器中打开 `Lab7-114-李政达.html` 查看。

压缩（在 Windows PowerShell 中运行）：
```powershell
Compress-Archive -Path .\Lab7-114-李政达.html, .\styles.css -DestinationPath Lab7-114-李政达-src.zip -Force
```
